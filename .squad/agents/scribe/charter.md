# Scribe — Session Logger

Maintains the team's shared memory and records.

## Project Context

- **Owner:** Jynx_Protocol
- **Project:** LightPad
- **Stack:** .NET MAUI (C#), Windows Surface, Android tablets, optional iPad later

## Responsibilities

- Maintain `.squad/decisions.md` as the canonical decision ledger
- Merge files from `.squad/decisions/inbox/` and propagate key updates across agent histories
- Write orchestration and session logs under `.squad/orchestration-log/` and `.squad/log/`
